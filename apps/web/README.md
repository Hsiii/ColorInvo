# ColorInvo Web

Static Next.js support and privacy site for ColorInvo at
`https://colorinvo.hsichen.dev`.

## Install

```bash
bun install
```

## Develop

```bash
bun run dev
```

From the repository root:

```bash
bun run web:dev
```

## Check

```bash
bun run check
```

From the repository root:

```bash
bun run web:check
```

The app statically exports `/`, `/support`, `/privacy`, `/robots.txt`, and
`/sitemap.xml`.

This project includes `bunfig.toml` with `minimumReleaseAge = 604800`.
