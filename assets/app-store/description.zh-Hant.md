# 條色盤 App 描述

## 副標題

把手機載具放進主畫面，配色跟桌布搭起來。

## App Store 描述

條色盤讓你的台灣電子發票手機條碼載具更好找，也更適合放在 iPhone 主畫面。

輸入手機載具後，條色盤會產生可掃描的 Code 39 條碼，並讓你先在 App 裡確認顯示效果。你可以使用經典黑白配色，也可以選一張喜歡的桌布圖片，讓 App 從圖片抓出代表色並推薦適合的條碼配色。儲存後，把小工具加到主畫面，就能在需要刷載具時快速亮出條碼。

條色盤不只是把條碼變漂亮。每組推薦與自訂配色都會依掃描器友善的對比規則檢查，降低因為顏色太接近而掃不到的機率。載具號碼、配色與小工具設定都保存在裝置上的 app group，App 和 Widget 會共用同一份設定，不需要帳號，也不需要把你的載具資料傳到伺服器。

主要特色：

- 儲存台灣電子發票手機條碼載具
- 在 App 內預覽載具條碼與小工具顯示
- 從桌布圖片自動產生搭配配色
- 支援自訂條碼與背景顏色
- 檢查配色是否符合掃描器友善的對比條件
- 使用 iOS 小工具快速顯示載具
- 載具與配色資料保存在裝置上

條色盤適合想把手機載具固定在主畫面、又希望條碼不要破壞桌布風格的人。設定一次之後，需要結帳刷載具時，就不用再翻找其他 App。
