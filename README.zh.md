## 功能介紹

- **一秒出示載具：** 利用 iOS 小工具輕鬆展示載具條碼。
- **桌布主題配色：** 選一張桌布圖片，自動抓主色並產生配色方案。
- **保證能掃的到：** 確保條碼配色符合商業掃描器反射率對比規則。
